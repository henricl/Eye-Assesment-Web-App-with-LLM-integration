# Eye Assessment Web App with LLM Integration

AI-powered web application for assessing eye symptoms using large language models and safety-oriented reasoning.

This project was developed as a prototype during my Computer Science Bachelor thesis.

---

## Overview

The application allows users to describe eye-related symptoms and receive structured guidance generated by AI models.

The system integrates multiple safety layers to reduce the risk of unsafe medical advice.

The goal is to explore how LLM-based systems can assist with symptom triage while maintaining responsible AI behavior.

---

## Features

- Conversational symptom input
- AI-based reasoning using large language models
- Safety architecture to reduce unsafe medical advice
- Web interface for user interaction
- Node.js backend with Firebase functions
- Integration with AI APIs for reasoning and classification

---

## Tech Stack

- Node.js
- Firebase Functions
- HTML / CSS
- Azure AI
- Hugging Face models
- JavaScript
